AAA Testing

AAA testing is a way to design test cases. AAA testing stands for Arrange-Act-Assert.

  • Arrange: Sets up the test case. For example, an object needs to be instantiated for the test case.
  • Act: Cover a section of the code to be tested. This can be a method or a function.
  • Assert: Checks the result of the response is the correct numeric or string value.
    • Asserts decides if the test passes.

Unit Testing of CSV files

Example

In this example,

  1. You got to import the pandas library.
  2. There is a function in pandas to read in a csv file. The function is called read_csv() that takes the file path as the parameter. The path must be an absolute path of the repository. Relative paths will not work. I set the read_csv() table to the dataframe variable.
  3. The name of the columns of the table are value_1, value_2 and result.
  4. There is got to be a way to read in the values of each column. The dataframe is a dictionary. I get the column data by specifying the column's name the dataframe should read into. Each column will be added into a tuple. This makes it easy for me to access the row of each value easily in a for loop.
  5. For the last step, I use a for loop to access the values of value1, value2 and the result of tuple. Each iteration checks if the addition of thoses two values gives the right result
